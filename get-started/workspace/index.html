    Workspace #

    Workspace is a folder in your machine disk that containing cherry studio record files, media files, … We only have one workspace used at a time.

    Initialize #

    1. Init workspace from empty folder

    Go to dashboard and click Init workspace . Then choose empty folder

    1. Init workspace from existing workspace folder

    Go to dashboard and click Init workspace . Then choose a folder having init.csow file inside.

    Detach #

    If you have multiple workspace, you can detach and init workspace again anytime. It’s harmless action because detaching won’t delete workspace file/folder in disk.
